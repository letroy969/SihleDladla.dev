< Online Student card creation system /> (Final-Year Group Project)

NB: Please note that the project might not have a database as our database has expired. For presentation purposes, we used a local database.
This was my final-year group project, developed collaboratively and presented as part of our Software Development program. Throughout the project lifecycle, we focused heavily on continuous testing, improvements, and refining the system based on real student challenges.
Problem
The chaotic and inefficient process students faced when obtaining their physical student cards. At the start of every academic year, thousands of students from all faculties had to queue in a single venue to get their cards printed. The system often crashed due to high load, queues stretched for hours, and many students even missed classes because of the long waiting times.
Approach
Built a secure, web-based Student Card System that allows students to register online, upload required documents that must be validated by system before approval, upload picture or use the live photo capture option on your device which the system must detect your face and store the image and generate a virtual student card instantlyeliminating the need for physical queues. Security was the highest priority, so we implemented proper authentication measures, validation standards, and verification processes to ensure data accuracy and system integrity. This was developed as a final-year group project with continuous testing, improvements, and refining based on real student challenges.
Result
To solve this, we built a secure, web-based Student Card System that allows students to register online, upload required documents, and generate a virtual student card instantlyeliminating the need for physical queues. Security was the highest priority, so we implemented proper authentication measures, validation standards, and verification processes to ensure data accuracy and system integrity.
Additional Benefits
An additional benefit is that long-distance or online-learning students can now access institutional resources easily without having to travel to campus for a physical card, also students get to keep a picture they like on their student card that is in compliance with the University photo standard.
This project demonstrated our ability to build a real-world, scalable solution that improves student experience while maintaining strong security and performance standards.

< AI Cybersecurity Honeypot />

This project is an AI-powered honeypot designed to safely analyze attacker behavior without risking real systems. It simulates vulnerable endpoints, captures malicious activity, and records detailed attacker telemetry for analysis.
Problem
Organizations needed a safe way to study attacker behaviors and improve security defenses without exposing real systems to threats.
Approach
Using machine learning models such as Isolation Forest and Autoencoders, the system classifies abnormal behavior, detects intrusion patterns, and predicts potential threat actions. This creates a controlled environment for studying real-world attack techniques and improving defensive strategies.
Result
The outcome is a practical, educational cybersecurity platform that provides meaningful insights into attacker behavior, anomaly detection, and automated threat analysis.
